Soluzione Pubblica Prova Finale API 2024
****************************************

AVVERTENZE:
 - Gli eseguibili in oggetto sono forniti solo come convenienza.
 - Garantiamo che la soluzione produce esattamente gli output richiesti dal
   verificatore per i casi di test previsti per la valutazione del corso. Non
   viene fornito nessun altro tipo di garanzia sulla correttezza degli output
   e sulla compatibilità degli eseguibili con la vostra macchina.
 - Le caratteristiche di performance di questa soluzione sono volutamente non
   ottimali.
 - Sono state prese misure precauzionali di natura tecnica per ostacolare
   l'ispezione del codice negli eseguibili.

Per eseguire la soluzione, seguire le istruzioni specifiche per il vostro
sistema operativo.

# Windows

Sono supportati Windows 10 e le versioni successive per processori Intel a
64 bit.

1. Aprire il prompt dei comandi (cmd.exe, non Powershell)
2. Digitare "cd" (senza virgolette) seguito da uno spazio e dal percorso dove
   si trova questo file. Per esempio, se questo file si trova nella cartella
   Download, e la vostra cartella utente è C:\Users\Studente,
   dovete digitare "cd C:\Users\Studente\Download".
3. Digitare "sol_2024_w64.exe" per eseguire la soluzione. I comandi vengono
   letti dal flusso "standard input" e gli output vengono scritti sul flusso
   "standard output".
   
Per inviare i comandi contenuti in un file, utilizzare gli operatori di
reindirizzamento. Per esempio, se avete un file di input chiamato "esempio.txt",
il comando da digitare è:
  sol_2024_w64.exe < esempio.txt
   
# Linux

È supportata la distribuzione Ubuntu 22.04 per processori Intel a 64 bit.

1. Aprire il Terminale della vostra distribuzione.
2. Digitare "cd" (senza virgolette) seguito da uno spazio e dal percorso dove
   si trova questo file. Per esempio, se questo file si trova nella cartella
   Downloads, e la vostra cartella utente è /home/studente,
   dovete digitare "cd /home/studente/Downloads".
3. Digitare "sol_2024_linux" per eseguire il generatore. I comandi vengono
   letti dal flusso "standard input" e gli output vengono scritti sul flusso
   "standard output".

Per inviare i comandi contenuti in un file, utilizzare gli operatori di
reindirizzamento. Per esempio, se avete un file di input chiamato "esempio.txt",
il comando da digitare è:
  sol_2024_linux < esempio.txt
   
# macOS

È supportato macOS Big Sur 11.0 e le versioni successive per processori Intel
a 64 bit e per processori Apple Silicon.

1. Aprire Terminale.app
2. Digitare "cd" (senza virgolette) seguito da uno spazio e dal percorso dove
   si trova questo file. Per esempio, se questo file si trova nella cartella
   Downloads, e la vostra cartella utente è /Users/studente,
   dovete digitare "cd /Users/studente/Downloads".
4. Digitare "xattr -c sol_2024_macos". Questo comando elimina l'attributo
   quarantena aggiunto automaticamente ai file scaricati da Internet.
   Questo attributo impedirebbe l'esecuzione di eseguibili non firmati da Apple
   come quello da noi fornito.
   Siccome siamo tirchi non intendiamo pagare 100$ a Apple per firmare un
   singolo eseguibile.
3. Digitare "sol_2024_macos" per eseguire il generatore. I comandi vengono
   letti dal flusso "standard input" e gli output vengono scritti sul flusso
   "standard output".

Per inviare i comandi contenuti in un file, utilizzare gli operatori di
reindirizzamento. Per esempio, se avete un file di input chiamato "esempio.txt",
il comando da digitare è:
  sol_2024_linux < esempio.txt
